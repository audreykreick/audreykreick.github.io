Absolute vs Relative CSS Length Units

The term absolute when talking about CSS length units means that the units are not relative to anything else. They are generally considered to always be the same size. Most of absolute units are more useful when used for print, instead of screen output. Relative lenth units, on the other hand, are relative to something else. A benefit of using relative units is that you can make it so that the size of text or other elements scales are relative to everything else on the page. Specific Length Units and What They Are

ch

  • Represents the width or, more specifically, the advance measure of the glyph 0 (zero, the Unicode character U+0030) in the element's font. In cases where determining the measure of the 0 glyph is impossible or impractical, it is assumed to be 0.5em wide by 1em tall.
